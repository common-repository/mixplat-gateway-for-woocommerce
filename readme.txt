Подключите оплату картами, СБП, Yandex.Pay и мобильные платежи на вашем сайте через Миксплат.

== Description ==

Плагин для приема платежей на сайте под управлением Wordpress + WooCommerce с помощью банковских карт (интернет-эквайринг), СБП, Yandex.Pay и мобильных платежей.

**Описание**

* 4 способа оплаты: банковские карты, СБП, Yandex.Pay, мобильные платежи
* Подключение и абонентская плата — 0 ₽
* Комиссия — от 0.4%

**Возможности**

* Автоматическая смена статуса заказа после оплаты
* Интеграция с онлайн-кассой (фискализация по 54-ФЗ)
* Поддержка режима холдирования (двухстадийная схема оплаты)
* Частичные или полные возвраты через личный кабинет

Больше информации — на [сайте](https://mixplat.ru/).

**Порядок использования**

1. Оставить заявку на [сайте](https://mixplat.ru/) и заключить договор.
2. Получить доступ в личный кабинет.
3. Установить и настроить модуль согласно [инструкции](https://mixplat.ru/developers/wordpress-woocommerce-payment-module/).
